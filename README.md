# OptiTrade Copilot: An AI-Driven Trading Portal with Interactive Dynamic Canvas

Our project is an AI-powered trading platform to help user in portfolio management and trading decisions. It has a
dynamic canvas, users can customize (drag-and-drop) layouts with AI-supported widgets. For example, a ‘news widget’
will come with AI news insights. A ‘price chart widget’ will be given with a price momentum analysis from AI. A
‘portfolio widget’ will include AI monitoring with event notification. Seamlessly integrating with multiple brokers,
the system provides full oversight of assets and enables trade execution directly. AI agents assist trading decision
called upon by the user, empowering traders with actionable insights to optimize strategies.
